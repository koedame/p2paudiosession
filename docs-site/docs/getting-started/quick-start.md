# クイックスタート

jamjamで最初のセッションを始める手順を説明します。

## 1. オーディオデバイスの設定

1. jamjamを起動
2. 設定画面を開く
3. 入力デバイス（マイク/楽器入力）を選択
4. 出力デバイス（ヘッドフォン推奨）を選択

:::tip
レイテンシを最小化するため、ASIOドライバ（Windows）またはオーディオインターフェースの専用ドライバを使用することを推奨します。
:::

## 2. ルームを作成する

1. 「ルーム作成」を選択
2. ルームIDが自動生成される
3. 招待コード（6文字英数字）または招待URLを共有

## 3. ルームに参加する

1. 「ルーム参加」を選択
2. 招待コードまたは招待URLを入力
3. 接続が確立されると音声通信が開始

## 4. 音声品質の調整

### プリセット

| プリセット | 用途 |
|-----------|------|
| ultra-low-latency | LAN環境での最小遅延 |
| balanced | 標準的なインターネット接続 |
| high-quality | 高音質優先 |

### 手動調整

- **サンプルレート**: 44.1kHz / 48kHz / 96kHz
- **フレームサイズ**: 64 / 128 / 256 / 512 / 1024 samples
- **コーデック**: PCM（無圧縮）/ Opus / FLAC

## トラブルシューティング

### 接続できない場合

- ファイアウォールでUDP通信が許可されているか確認
- NATタイプを確認（対称NATの場合、TURN中継が必要）

### 音声が途切れる場合

- フレームサイズを大きくする（例: 128 → 256）
- FEC（前方誤り訂正）を有効にする
