# クイックスタート

jamjamで最初のセッションを始める手順を説明します。

## 1. オーディオデバイスの設定

1. jamjamを起動
2. 画面右上の **設定**（歯車アイコン）を開く
3. **入力デバイス**（マイク/楽器入力）を選択
4. **出力デバイス**（ヘッドフォン推奨）を選択
5. **バッファサイズ**を調整（デフォルト: 256サンプル）

:::tip
レイテンシを最小化するため、ASIOドライバ（Windows）またはオーディオインターフェースの専用ドライバを使用することを推奨します。
:::

## 2. ルームを作成する（ホスト）

1. メイン画面で **「ルーム作成」** を選択
2. ルーム名を入力（任意）
3. パスワード保護が必要な場合は設定
4. **招待コード**（6文字英数字）が表示される
5. 招待コードを相手に共有

## 3. ルームに参加する（ゲスト）

1. メイン画面で **「コードで参加」** 入力欄に招待コードを入力
2. または **「最近の接続」** から過去に接続したルームを選択
3. パスワードが設定されている場合は入力
4. **「参加」** を押すと接続開始

## 4. セッション中の操作

### 入力レベルの確認

- 画面上の **レベルメーター** で入力音量を確認
- 緑〜黄色が適正、赤はクリッピングの可能性

### ミュート機能

- **マイクアイコン** をクリックしてミュート/ミュート解除
- セッション中に一時的に音声を止めたい場合に使用

### 接続品質の確認

- **RTT**（往復遅延時間）と **ジッター** が表示される
- ジッター値に基づいた **推奨プリセット** が表示される

## 5. プリセットの選択

ネットワーク環境に応じて最適なプリセットを選択します:

| プリセット | 用途 | ジッター目安 |
|-----------|------|-------------|
| zero-latency | 超低遅延環境（専用線等） | < 1ms |
| ultra-low-latency | LAN環境 | 1-3ms |
| balanced | 標準的なインターネット接続 | 3-10ms |
| high-quality | 高音質優先、高遅延許容 | > 10ms |

:::tip
画面に表示される **「推奨設定に切り替え」** ボタンで、現在の接続品質に最適なプリセットに自動変更できます。
:::

## 6. 外観の設定

### ダークモード

1. 設定画面を開く
2. **テーマ** セクションで選択:
   - **ダーク**: 暗い配色
   - **ライト**: 明るい配色
   - **システム**: OSの設定に追従

### 言語設定

- 日本語 / English を選択可能

## 7. セッションの終了

1. **「切断」** ボタンを押す
2. または画面を閉じる

:::note
ホストが切断するとルームは削除されます。
ゲストは自由に切断・再接続が可能です。
:::
