# jamjam

ミュージシャン向け超低レイテンシP2P音声通信アプリ

## 概要

jamjamは、ミュージシャンがインターネット越しにリアルタイムでジャムセッションや遠隔レコーディングを行うためのアプリケーションである。

## ドキュメント構成

| [architecture.md](./architecture.md) | 技術構成（最重要） |
| [adr/](./adr/) | 設計判断記録 |
| [behavior/](./behavior/) | 振る舞い定義（BDD） |
| [api/](./api/) | API境界定義 |

## 対象ユースケース

- オンラインジャムセッション
- 遠隔レコーディング
- リアルタイム演奏セッション

## 対象プラットフォーム

- Windows
- macOS
- Linux
- iOS（将来）
- Android（将来）
