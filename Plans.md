## タスク作成ガイドライン

### 良いタスクの例

```markdown
- [ ] [impl] NoiseGate エフェクトを追加
  - threshold, attack, release パラメータ対応
  - 参照: docs-spec/api/audio_engine.md
```

### タスクの種類

| プレフィックス | 説明 |
|--------------|------|
| `[impl]` | 実装タスク |
| `[fix]` | バグ修正 |
| `[refactor]` | リファクタリング |
| `[docs]` | ドキュメント更新 |
| `[test]` | テスト追加 |
| `[infra]` | インフラ・CI/CD |
